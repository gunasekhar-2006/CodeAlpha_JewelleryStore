# LuxeJewels - MERN Stack Jewellery E-Commerce Website

LuxeJewels is a modern jewellery e-commerce website developed using the MERN Stack.  
This project was created as part of my internship project to practice full stack web development, authentication, API integration, database handling and deployment.

---

## Live Demo

Frontend: https://code-alpha-jewellery-store.vercel.app/

Backend API: https://code-alpha-jewellery-store.vercel.app/

GitHub Repository: https://github.com/gunasekhar-2006/CodeAlpha_JewelleryStore

---

## Features

- User Registration and Login
- Jewellery Categories
- Product Search Functionality
- Wishlist Management
- Buy Now Functionality
- Order History
- Cancel Order Feature
- Product Details Page
- Responsive UI Design
- MongoDB Database Integration
- Full Stack Deployment

## Installation

### Clone Repository

```bash
git clone https://github.com/gunasekhar-2006/CodeAlpha_JewelleryStore.git
```

### Frontend Setup

```bash
cd client
npm install
npm run dev
```

### Backend Setup

```bash
cd server
npm install
npm run dev
```
